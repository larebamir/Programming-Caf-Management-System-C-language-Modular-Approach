# Programming Café Management System

The Programming Café Management System is a command-line application that allows users to manage tutors, sessions, and students for a programming café. It provides functionalities for adding tutors, sessions, and students, as well as enrolling students into sessions and listing the assigned sessions for each student.

## Files

- `main.c`: The main source code file containing the implementation of the application's functionalities.
- `menu.h`: The header file defining the menu functions for different user roles.
- `session.h`: The header file defining the structure and functions related to sessions.
- `tutor.h`: The header file defining the structure and functions related to tutors.
- `student.h`: The header file defining the structure and functions related to students.
- `constants.h`: The header file containing constant values used in the application.
